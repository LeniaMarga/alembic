Alembic Layout

StackLayout adds whitespace between elements along the vertical axis.

BoxLayout is a basic primitive to show a container of some elements. It has a padding and borderWidth to configure the box. It can be inverted with invert to swap the text and background colour.

CenterLayout ensures a block-level element is placed in the horizontal middle of its parent. The element will be at most max width and spacing will be added at either side. An optional gutters can be applied to add padding to either side of the cantered element. It can also use the intrinsic width of the centered element so inline elements will be perfectly centered (e.g. buttons).

ClusterLayout groups multiple items together based on their intrinsic width and seperated by a gutter decided by the space prop.

SidebarLayout places two elements side-by-side if space permits it or stacks them on top of each other if not. The sidebar can either be a set width or it's intrinsic width (based on the width of the content inside it) and the content element will fill up the rest of the horizontal space. When there is not enough space (e.g. on mobile) where the content is less than contentMin percentage, the sidebar and content stack vertically.

SwitcherLayout dynamically displays a set of columns horizontally if there is enough space or vertically if not. It is based on threshold which is the minimum width that it will show elements horizontally. It has a limit where if exceeded elements will always display vertically.

GridLayout creates a responsive CSS Grid where each element has a min width. Element start at the minimum width and grow proportionally until all elements fill the layout horizontally or there is space for "ghost" elements. Each element has a gutter decided by the space prop.

FrameLayout ensures it's child element has a fixed aspect ratio and centers any content if not an image.

ReelLayout places elements horizontally with a set space between them and ensures overflow scrolling happens. Unless noBar is set, the ReelLayout will show scrollbars to indicate that there is horizontally scrollable content. Set the height to force all elements to be a set height, otherwise it will use the elements intrinsic height (within a Flexbox context).

ImposterLayout positions an element over other elements. The positioned element is vertically and horizontally centered in it's nearest positioned parent and will not exceed it's height or width. It can optionally be fixed to the viewport instead. The width of the element is 50% of the positioned parent unless overridden in CSS, it is setup to not exceed 100% (minus margins).
