# ⚗️ Alembic

A Design System for Open Lab research projects.

[**Go to the docs →**](https://digitalinteraction.github.io/alembic/)

## Release process

1. Run the build
2. Update API docs if needed
3. Run `npm version minor|major|patch`
4. Run `npm publish`
