Layouts

Layouts are custom elements that's only responsibility is to responsively position and arrange its child elements. These elements can be reused and nested as much as you like to compose pages together.

They share a common set of css variables that need to be set on your :root element, see a specific version for more info.

Scale variables. These are created using a modular scale (more info) to work nicely with a font size of 1rem and line-height of 1.5. Think of them as starting at -5 going though 0 and up to 5. They form a curve of values which creates pleasing spacing. These are provided by Alembic.

  • --s-5 — Smallest value
  • --s-4
  • --s-3
  • --s-2
  • --s-1
  • --s0
  • --s1 — Common default scale
  • --s2
  • --s3
  • --s4
  • --s5 — Largest largest
