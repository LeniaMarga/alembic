{{ icon('info') }} About

Alembic is a Design System built with Modern CSS based on Progressive Enhancement to deliver accessible experiences on the web. It is meant to work with the browser, not fight it, embracing HTML and CSS & minimising JavaScript.

Alembic is a living project and a work in progress. Inspired by CUBE CSS, EveryLayout, Bulma and Form design patterns.
